Features

General

  • Load and Save IM library image formats:
    PNM, ICO, PNG, JP2, BMP, PCX, TIFF, JPEG, GIF, RAS, TGA and SGI.
  • Load and Save multipage formats as TIFF and GIF, or frames as AVI and WMV in Windows.
  • Import and Export RAW data
  • Import and Exports CGM, WMF, EMF
  • Export image as EPS and SVG
  • Print support in Windows
  • Processing Log (a simple History)
  • Windows Clipboard Copy and Paste
  • Capture from cameras using Direct X in Windows

View

  • Zoom
  • Additional Bitmap views
  • Histogram
  • Image Matrix
  • 3D View of 2F image data
  • Full Screen
  • Real 2 Integer conversions
  • Complex 2 Real conversions
  • Hues palette for viewing angles
  • Mouse coordinates information on the Status Bar
  • Results Matrix

Image

  • Byte, Unsigned Short, Integer, Real and Complex data types
  • XYZ, Lab, YCbCr, CMYK, RGB, Gray Scale, Binary and RGB Palette Indexed color modes
  • Generic Attributes (metadata)
  • Gray images pseudo coloring
  • Undo when operation over the same image window
  • Counter to follow/interrupt long time processing

Processing / Geometric

  • Resize by Bilinear, Bicubic and Zero Order interpolation
  • Reduce by Bilinear and Zero Order decimation
  • Fast reduce are by 2 and by 4
  • Crop, Add Margins  
  • Free Rotate, Rotate 90/180, Flip, Mirror
  • Radial Distortion

Processing / Quantize

  • Reduce Grays Uniform and Median Cut
  • RGB to 216 Colors w/wo Dithering
  • RGB to <256 Colors using Median Cut

Processing / Color Components

  • RGB to Gray Scale using Luma and Linear weights
  • Split/Merge RGB
  • Split/Merge HSI  
  • Split Chroma/Intensity
  • Plot Yrb Ghroma Gamut
  • Plot HS Gamut 

Processing / Threshold

  • Manual, by Percentage
  • Diffusion and Uniform Error
  • by Local Contrast, by Local Max
  • by Difference
  • Otsu, Hysteresis

Processing / Histogram

  • Equalization and Expansion

Processing / Render

  • Constant, Color, Box, Tent, Cone, Wheel, Gaussian, Cosine, Sinc, Ramp 
  • Add Uniform/Speckle/Gaussian Noise

Processing / Unary Arithmetic

  • Inverse, Less, Square, Square Root, Exponential,
    Logarithm, Sine, Cosine, Absolute
  • Split/Merge Complex, Conjugate
  • Normalize Complex by Magnitude 

Processing / Tone Gamut

  • Normalize, UnNormalize: Conversions (0-1) <-> (0-255)
  • Normalized Power (Gamma), Log and Exp
  • Slice, Expand, Zero Start 
  • Tone Crop
  • Brightness and Contrast 

Processing / Arithmetic

  • Add, Subtract, Multiply, Divide, Power, Maximum, Minimum, Difference
  • Blend
  • Background Subtraction
  • Mean and StdDev of multiple images

Processing / Logical Arithmetic

  • And, Or, Xor, Not
  • Bit Plane Extract and Reset 

Processing / Filter

  • Kernel Convolution
  • Image Convolution
  • Compass
  • Gaussian, Circular Mean
  • Difference of Gaussians
  • Laplacian of Gaussian 
  • Sobel, Zero Crossing
  • Canny

Processing / Filter (Rank)

  • Median
  • Closest MinMax, Local Max, Local Min, Range (Max-Min)

Processing / Morphology (Binary)

  • Kernel Convolution
  • Erode, Dilate, Open, Close
  • Outlining, Thinning  

Processing / Morphology (Gray Scale)

  • Kernel Convolution
  • Erode, Dilate, Open, Close
  • Top Hat, Well, Gradient

Processing / Binary

  • Prune, Fill Holes 
  • Perimeter Line
  • Regional Maximum

Processing / Transform

  • FFT, IFFT,Swap Quadrants
  • Hough Lines
  • Cross Correlation
  • Auto Correlation
  • Distance Transform

Processing / Effects

  • Negative, Solarize
  • Pixelate, Posterize
  • Replace RGB, Replace Constant

Analyze / Statistics

  • Count Colors
  • Image Statistics
  • Histogram Statistics
  • RMS Error, SNR

 Analyze

  • Find Regions
  • Measure Regions:
    Area, Perimeter, Holes, Centroid, Principal Axis
